Firstly my project's interface will be like this

Now after we Login to our Account Dashboard will be apper as a active page. There we can find many options like

Now we have to click on initialize bot option so our chatbot will start and we can ask any question to that bot we will get the answer in fraction of seconds. It very helpful to farmers
